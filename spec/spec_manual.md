# Requirement ID: FR1

- Description: The system shall provide users with supportive and relevant responses to help manage anxiety and negative thoughts.
- Source Persona: Emma
- Traceability: Derived from review group G1
- Acceptance Criteria: Given the user expresses feelings of stress or anxiety When the system generates a response Then the response must include supportive language and at least one coping suggestion.
